POS System (PHP & MySQL)

- A simple and efficient Point of Sale (POS) system built with PHP, MySQL, Bootstrap, and JavaScript.
This project helps manage products, categories, customers, and orders with a clean dashboard and analytics.

Features
- Authentication (Login system)
- Dashboard with statistics (Orders, Customers, Products)
- Create and manage orders
- Category management (Add, Edit, Delete)
- Product management (Add, Edit, Delete)
- Customer management (Add, Edit, Delete)
- Admin/Staff management (Add, Edit, Delete)
- chart (Orders & Customers overview)
- Search by date and payment method for orders
- Print and Download the invoice

Technologies Used
- Frontend: HTML, CSS, Bootstrap 5, JavaScript
- Backend: PHP (Core PHP)
- Database: MySQL
- Charts: Chart.js
